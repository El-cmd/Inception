    Ces étapes expliquent comment préparer et installer Docker sur un système Debian.

    1. Mise à jour des paquets et installation des prérequis

      sudo apt-get update && sudo apt-get install apt-transport-https ca-certificates curl gnupg2 software-properties-common
      • sudo apt-get update : Met à jour l'index des paquets.
      • sudo apt-get install ... : Installe les paquets nécessaires pour télécharger via HTTPS et gérer les clés de sécurité.

    2. Ajout de la clé GPG du dépôt officiel de Docker

      curl -fsSL https://download.docker.com/linux/debian/gpg | sudo apt-key add -
      • curl -fsSL [URL] : Télécharge la clé GPG de manière sécurisée.
      • sudo apt-key add - : Ajoute la clé au système de gestion des paquets.

    3. Ajout du dépôt Docker à la liste des sources de paquets

      sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/debian $(lsb_release -cs) stable"
      • Ajoute le dépôt de Docker pour Debian pour la branche stable et la version spécifique du système.

    4. Mise à jour des paquets et installation de Docker

      sudo apt update && sudo apt-get install docker-ce docker-ce-cli containerd.io
      • sudo apt update : Actualise l'index des paquets après l'ajout du nouveau dépôt.
      • sudo apt-get install ... : Installe Docker et ses composants essentiels.
